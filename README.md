A react app:

git clone <your-repo-url>
cd your-project-folder

npm install

Setup the backend:

cd server
npm init -y
npm install express axios cheerio cors

Start the backend:

cd server

node (or nodeom) index.js

Start React Frontend:

cd ..
npm start
